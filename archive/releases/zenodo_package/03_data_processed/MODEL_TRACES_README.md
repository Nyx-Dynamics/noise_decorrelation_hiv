# Model Trace Files Summary

## Overview

This document summarizes the important MCMC trace files (NetCDF format) generated by the Bayesian models used in the manuscript "Noise Decorrelation as a Hypothetical Mechanism for Phase-Specific Neurometabolic Outcomes in HIV Infection."

All traces were generated using PyMC 5.x with the NUTS (No-U-Turn Sampler) algorithm and stored in ArviZ-compatible NetCDF format.

## Trace File Format

All `.nc` files are NetCDF4 format compatible with ArviZ. They contain:

1. **posterior** - MCMC samples for all parameters
2. **posterior_predictive** - Predicted observations
3. **sample_stats** - Sampler diagnostics (divergences, tree depth, etc.)
4. **observed_data** - Original input data

### Loading Traces in Python

```python
import arviz as az

# Load trace
idata = az.from_netcdf("trace_v3_6.nc")

# Summary statistics
print(az.summary(idata))

# Plot posteriors
az.plot_posterior(idata, var_names=["xi_nm_acute", "xi_nm_chronic", "beta_xi"])

# Diagnostics
print(f"R-hat max: {az.rhat(idata).max()}")
print(f"ESS min: {az.ess(idata).min()}")
```
